# Discord チャンネル名自動更新ボット

## 概要

このDiscord Botは、特定のチャンネル（チャンネルA）で5桁の数字を含むメッセージを検出し、以下の2つの処理を行います：

1. 受け取ったメッセージ（5桁の数字）を別のチャンネル（チャンネルB）に転送。
2. チャンネルBの名前に含まれる `【xxxxx】` の部分を、受け取った5桁の数字に置換してチャンネル名を更新。

## 使用技術

- Python 3.x
- [discord.py](https://github.com/Rapptz/discord.py) ライブラリ

##  動作イメージ

- チャンネルAで `12345` と送信すると、
  - チャンネルBに `12345` と投稿される。
  - チャンネルBの名前が `チャンネル名【12345】` のように変更される（既存の【xxxxx】部分が置換される）。

## 使用方法

### 1. 依存ライブラリのインストール

```bash
pip install discord.py
```

### 2.BOTのトークンを設定

bot.run('あなたのBotトークン')

### 3. チャンネルIDの設定
CHANNEL_A_ID = あなたのチャンネルAのID
CHANNEL_B_ID = あなたのチャンネルBのID

### 4.実行
python bot.py

# 注意点
チャンネル名の変更には Botに「チャンネル管理」権限 が必要です。

チャンネルBの名前には、最初から 【xxxxx】 形式の5桁数字が含まれている必要があります。

トークンを公開リポジトリに載せないでください！(記載のものはサンプル)
